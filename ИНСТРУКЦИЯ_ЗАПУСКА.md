# 🏃‍♂️ Инструкция запуска SportChat 

## 📋 Требования
- Python 3.8+
- Node.js 16+

## 🚀 Быстрый запуск

### 1. Backend (FastAPI)
```bash
cd backend

# Установка зависимостей (только при первом запуске)
pip install -r requirements.txt

# Запуск сервера разработки
uvicorn main:app --reload --port 8000
```
✅ **Backend доступен на: http://localhost:8000**  
📚 **API документация: http://localhost:8000/docs**  
🔍 **Health check: http://localhost:8000/health**

**Возможные проблемы:**
- `Address already in use` - Backend уже запущен или порт занят
- `ModuleNotFoundError` - Установите зависимости: `pip install -r requirements.txt`
- Database errors - SQLite база создается автоматически при первом запуске

### 2. Frontend (React + shadcn/ui)
```bash
cd frontend
npm install
npm start
```
✅ Frontend доступен на: **http://localhost:3000**

## 🔧 React Hooks - Важные правила разработки

### Правила работы с useEffect:
```typescript
// ✅ ПРАВИЛЬНО: Включаем все зависимости
useEffect(() => {
  loadChatHistory();
}, [userId, loadChatHistory]);

// ✅ ПРАВИЛЬНО: Используем useCallback для стабильности
const loadChatHistory = useCallback(async () => {
  try {
    const history = await chatApi.getChatHistory(userId);
    setMessages(history);
  } catch (error) {
    console.error('Error loading chat history:', error);
  }
}, [userId]);

// ❌ НЕПРАВИЛЬНО: Отсутствуют зависимости
useEffect(() => {
  loadChatHistory(); // ESLint warning
}, [userId]); // Функция loadChatHistory не включена
```

### Правила управления состоянием:
```typescript
// ✅ ПРАВИЛЬНО: Используем функциональные апдейты
setMessages(prev => [...prev, newMessage]);

// ✅ ПРАВИЛЬНО: Мемоизируем тяжелые вычисления
const expensiveValue = useMemo(() => {
  return messages.reduce((sum, msg) => sum + msg.length, 0);
}, [messages]);

// ❌ НЕПРАВИЛЬНО: Прямая мутация состояния
messages.push(newMessage); // Никогда не мутируйте состояние напрямую
```

### ESLint правила:
- Всегда включайте все зависимости в массив зависимостей useEffect
- Удаляйте неиспользуемые импорты и переменные  
- Используйте useCallback для функций в зависимостях useEffect
- Мемоизируйте тяжелые вычисления с useMemo
